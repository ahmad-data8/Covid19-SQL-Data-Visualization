# COVID-19 Global Insights (SQL & Data Visualization)

## Project Overview
This project showcases the integration of relational databases (SQL) and data visualization within a Python environment. Using **SQLite3** and **Matplotlib**, I engineered a local database pipeline to store pandemic logs, query filtered subsets, and generate visual graphical reports.

## Core Features
1. **Relational Database Setup:** Built a temporary SQL database in-memory, defining schemas (`CREATE TABLE`) and populating records (`INSERT INTO`).
2. **SQL Query Engine:** Executed relational algebra queries to filter complex criteria, such as extracting countries with recovery rates exceeding 95%.
3. **Data Visualization:** Developed a custom bar chart reporting engine using Matplotlib to graphically illustrate case distributions across regions.

## Tech Stack
* **Language:** Python
* **Database:** SQLite3 / SQL
* **Visualization Library:** Matplotlib
* **Data Frame Handling:** Pandas
